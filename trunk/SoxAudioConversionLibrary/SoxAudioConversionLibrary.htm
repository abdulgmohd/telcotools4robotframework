Setting Value
Documentation Resource file of custom user keywords acting as library for common audio conversion routines using SoX application (http://sox.sourceforge.net/ ). Requires SoX application to be installed and use of the Robot Framework standard Operating System library.
Library OperatingSystem

 

Variable Value Value
${soxPath}= sox #default of local path, change path to SoX application as needed here. Path may differ between Windows and *nix installations.

 

Keyword Action Argument Argument
Convert Wave File For Dtmf And Tone Detection [Arguments] ${inputFile} ${outputFile}
  [Documentation] Convert given input audio file with SoX to an output audio file in the following format for use in telecom testing (e.g. DTMF or tone frequency detection):\n\n8 kHz, 16-bit, mono, (ideally) PCM encoding.  
  Run ${soxPath} ${inputFile} -c 1 -b 16 -r 8000 ${outputFile}
